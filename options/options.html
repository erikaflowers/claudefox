ClaudeFox Settings

Stored locally in Firefox sync storage. Only sent to api.anthropic.com.
Shapes how Claude responds — tone, style, perspective. This wraps around every mode prompt (Summarize, ELI5, etc.) like CLAUDE.md does for Claude Code. Leave blank for default Claude voice.

About ClaudeFox

ClaudeFox is a free, open-source community project. No tracking, no telemetry, no data collection. Your API key stays on your machine and page content is sent only to Anthropic's API.

Built and maintained by Erika Flowers with contributions from the community.
